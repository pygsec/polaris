## 前言

前因编程书籍之良莠不齐，又见博文分享之杂乱无章，故愤而编此册。  

余深感道阻且长，故披荆斩棘于月下，历半载之功，初见端倪，虽形神俱疲，但乐在其中。   

余亦深虑己之不足，而学无涯，尚有大量文章等待校正，余必晨兢夕厉，持之以恒。
